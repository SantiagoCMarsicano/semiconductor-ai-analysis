# Semiconductor Industry and Artificial Intelligence

## Project Overview

This project analyzes the evolution of the global semiconductor industry between 2010 and 2025, with special emphasis on the impact of artificial intelligence on industry growth and market valuation.

## Objectives

- Analyze the evolution of global semiconductor sales.
- Study the growth of AI-related demand.
- Compare industry trends with the performance of leading firms.
- Produce visualizations and an analytical report using R.

## Tools

- R
- tidyverse
- ggplot2
- GitHub
